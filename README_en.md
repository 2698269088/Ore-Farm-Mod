# Ore Farm Mod

A Minecraft 1.12.2 mod that adds ore farming capabilities to the game.

## Overview

Ore Farm is a mod that allows players to grow ores as crops. Instead of traditional mining, players can now farm various ores by planting special seeds and growing them like regular crops. This adds a new gameplay dimension to Minecraft, combining agriculture with resource gathering.

## Features

### New Items
- Poor Diamond - A low-grade diamond that can be crafted into regular diamonds
- Diamond Shards - Used to craft Poor Diamonds
- Emerald Shards - Used to craft Emeralds
- Various plant seeds for different ores:
  - Diamond Shard Plant Seed
  - Emerald Shard Plant Seed
  - Coal Shard Plant Seed
  - Iron Shard Plant Seed
  - Gold Shard Plant Seed
  - Redstone Shard Plant Seed
  - Lapis Lazuli Shard Plant Seed
  - Diamond Ore Seed
  - Emerald Ore Seed
  - Coal Ore Seed
  - Iron Ore Seed
  - Gold Ore Seed
  - Redstone Ore Seed
  - Lapis Lazuli Ore Seed

### New Blocks
- Poor Diamond Ore - A mineable block that drops Poor Diamonds
- Various crop blocks for growing ore-related items:
  - Diamond Crop
  - Emerald Crop
  - Coal Crop
  - Iron Crop
  - Gold Crop
  - Redstone Crop
  - Lapis Crop
  - Diamond Plant
  - Emerald Plant
  - Coal Plant
  - Iron Plant
  - Gold Plant
  - Redstone Plant
  - Lapis Plant

## Crafting Recipes

### Basic Crafting
- 9 Diamond Shards = 1 Poor Diamond
- 9 Emerald Shards = 1 Emerald

### Seed Creation
Each seed type can be crafted using a specific pattern with wheat seeds and the corresponding ore/ingot/nugget.

### Smelting
- Poor Diamond = 1 Diamond (in furnace)

## Farming Process

1. Craft the seeds you want to plant
2. Plant seeds on farmland like regular crops
3. Wait for them to grow through 8 stages (0-7)
4. Harvest fully grown crops to get the corresponding items
5. Use the harvested items for crafting or smelting

## Installation

1. Download and install Minecraft 1.12.2
2. Download and install Minecraft Forge 1.12.2
3. Download the Ore Farm mod .jar file
4. Place the .jar file in your Minecraft mods folder
5. Launch Minecraft with the Forge profile

## Compatibility

This mod is compatible with:
- Minecraft 1.12.2
- Forge 14.23.5.2860
- JEI (Just Enough Items) - for viewing recipes

## Known Issues

- Some recipes may not be visible in JEI if there are mod conflicts
- Models may not load properly if texture files are missing

## Building from Source

To build this mod from source code:
1. Clone the repository
2. Run `./gradlew build` in the project directory
3. The built .jar file will be located in the `build/libs` folder
